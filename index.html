Introduction

Tonba is a revolutionary redesign of Bitcoin that aims to standardize the new generation of digital decentralized currency. Using the Proof of Fortune concept, Tonba aims to create a greener mining system, as mining a block no longer relies solely on a user's computing resource. Project Tonba is continuously being improved on, with new enhancements and features constantly rolling out. Stay tuned for more updates!

Features

  • Proof of Fortune
  • Green Mining
  • More powerful and extensible payment scheme
  • Capital risk-free pooled mining
  • Secure thin client
  • Upgradable Protocol
  • And much more ...
